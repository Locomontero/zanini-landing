# Zanini Laser Landing Page

Landing page desenvolvida para a **Zanini Laser**, empresa especializada em serviços de **gravação a laser personalizada**.

O projeto foi construído com base no design criado no Figma e desenvolvido utilizando tecnologias modernas de frontend.

## 🎨 Design

O layout original pode ser acessado no Figma:

https://www.figma.com/design/3daZoaR2HBIWyB9fTFcAX8/ZaniniLaser

## 🚀 Tecnologias utilizadas

- React
- TypeScript (TSX)
- JavaScript
- Vite
- CSS
